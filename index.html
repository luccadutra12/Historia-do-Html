História do HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas da web. Ela foi desenvolvida por Tim Berners-Lee em 1991 e tem evoluído ao longo dos anos para se tornar a base da construção de sites modernos. O HTML permite estruturar o conteúdo da web, incluindo texto, imagens, links e outros elementos, tornando possível a criação de páginas interativas e visualmente atraentes.

Criador do HTML

Tim Berners-Lee é o criador do HTML. Ele desenvolveu a linguagem em 1991 enquanto trabalhava no CERN, o centro de pesquisa europeu para física de partículas.

Foto de Tim Berners-Lee

Principais versões do HTML

  • HTML 1.0 (1991): A primeira versão do HTML, criada por Tim Berners-Lee.
  • HTML 2.0 (1995): Introduziu novas tags e recursos, como formulários.
  • HTML 3.2 (1997): Adicionou suporte a scripts e applets.
  • HTML 4.01 (1999): Estabeleceu padrões para acessibilidade e internacionalização.
  • XHTML 1.0 (2000): Uma reformulação do HTML usando a sintaxe XML.
  • HTML5 (2014): A versão mais recente, com suporte a multimídia, gráficos e APIs avançadas.
